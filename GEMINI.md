# Проект "ФинТвист" – Ваш финансовый ассистент

## 8. Статус проекта (на 01.03.2026)

### 8.1. Достигнутый прогресс
*   **Инициализирован проект:** Создана структура папок, настроены зависимости (`requirements.txt`).
*   **Настроен CI/CD:** Проект выгружен на GitHub.
*   **Реализован MVP бота:**
    *   Бот запускается и отвечает на команду `/start`.
    *   Интеграция с **Ollama (Qwen 2.5 / Llama 3)** для распознавания трат.
    *   Распознавание **нескольких трат** в одном сообщении.
*   **Новые функции (Текущая сессия):**
    *   **Бюджетирование:** Команда `/budget` для установки лимитов. Автоматические уведомления при расходе 80% и 100% лимита.
    *   **Умные советы:** Команда `/advice` — LLM анализирует историю трат за месяц и дает персональные рекомендации.
    *   **Функция отмены:** Команда `/undo` удаляет последнюю ошибочно введенную трату.
    *   **Логирование ошибок:** Все сбои и таймауты теперь сохраняются в таблицу `error_logs` в БД для отладки.
    *   **Улучшенный парсинг:** Бот теперь понимает форматы типа "200р", "100 руб" и корректно обрабатывает вариативные ответы от LLM.
    *   **Стабильность:** Введен таймаут 2 минуты на запросы к ИИ и вежливое сообщение при перегрузке системы.
    *   **Консольный режим:** Обновлен `tests/terminal_bot.py` для тестирования всех функций без Telegram.

### 8.2. Следующие шаги
1.  **Визуализация:** Начать разработку графиков (диаграммы расходов) в Telegram через Mini Apps или отправку изображений.
2.  **Экспорт данных:** Команда для выгрузки всех трат в CSV/Excel.
3.  **Категории:** Возможность пользователя создавать свои категории.

### 8.3. Технические детали
*   **БД:** SQLite. Новые таблицы: `users` (с полем `monthly_budget`), `expenses`, `error_logs`.
*   **LLM:** Локальный Ollama.
*   **Язык:** Python 3.12+.
